# jspsych-starter
jsPsychを使った実験や調査のスタートラインとしてご利用ください

## jsPsychのダウンロード
1. [Tutorials -> The Basics: Hello World](https://www.jspsych.org/latest/tutorials/hello-world/)を開き、右側のメニューから[Option 2: Download and host jsPsych](https://www.jspsych.org/latest/tutorials/hello-world/#option-2-download-and-host-jspsych)を選択します。direct link (jspsych.zip) をクリックして、jsPsychをダウンロードしてください。

2. zipファイルを展開すると、distという名前のフォルダが見つかると思います。distフォルダと、ここで公開している jspsych-starter.html が同じフォルダ内に存在するようにファイルを配置してください。
